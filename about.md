# About

## 블로그 소개

이 사이트는 프로그래밍, 알고리즘, AI/LLM, 영어 학습 주제를 다루는 개인 기술 블로그입니다.  
핵심 목표는 "검색으로 찾은 정보를 다시 복붙"하는 것이 아니라, **직접 학습·실험한 내용을 이해하기 쉬운 구조로 정리**해 독자에게 실질적인 도움을 주는 것입니다.

## 작성자와 운영 방향

안녕하세요. 실무와 학습을 병행하며 개발/AI 분야를 연구하는 운영자입니다.

- **주요 주제**: Python, C, 알고리즘, LLM 애플리케이션, 기술 트렌드
- **콘텐츠 스타일**: 입문자 친화 설명 + 실무에서 바로 쓰는 기준 제시
- **GitHub**: [awithu](https://github.com/awithu)

## 어떤 콘텐츠를 제공하나요?

### 1) 프로그래밍/알고리즘 학습 시리즈
- 기초 문법부터 예제 중심으로 단계별 학습
- 단순 정의보다 "언제, 왜 사용하는지"를 함께 설명

### 2) AI/LLM 실무 글
- 프레임워크 비교, 아키텍처 선택 기준, 운영 팁
- 도구 간 차이를 계층 관점에서 설명해 의사결정에 도움

### 3) 영어 학습 콘텐츠
- 문법, 회화, 실전 표현을 목적별로 구성
- 꾸준히 학습할 수 있도록 시리즈 구조로 제공

## 콘텐츠 품질 기준

본 사이트는 다음 기준에 따라 글을 관리합니다.

1. **고유성**: 게시 글은 직접 작성하며, 자동 생성/복제 콘텐츠를 지양합니다.
2. **정확성**: 기술 문서, 공식 레퍼런스, 실습 결과를 교차 확인합니다.
3. **가독성**: 초보자도 따라갈 수 있도록 용어 설명과 예시를 포함합니다.
4. **업데이트**: 기술 변화나 오류 제보가 있으면 게시글을 수정/보완합니다.

편집/검수/수정 원칙은 [편집 정책](/editorial-policy/)에서 확인할 수 있습니다.
