# 편집 정책 (Editorial Policy)

최종 업데이트: 2026년 2월 11일

본 페이지는 이 사이트의 콘텐츠 제작 및 관리 원칙을 설명합니다.

## 1. 콘텐츠 작성 원칙

- 게시글은 운영자가 직접 작성합니다.
- 단순 복제/재가공/자동 대량생성 형태의 글을 지양합니다.
- 글의 목적(학습, 비교, 실무 적용)을 명확히 제시합니다.
- 제목과 본문 내용이 일치하도록 구성합니다.

## 2. 사실 확인과 출처 기준

- 기술 정보는 공식 문서, 공개된 레퍼런스, 실습 결과를 기준으로 검토합니다.
- 비교/해석이 포함된 글은 판단 기준과 전제를 본문에서 설명합니다.
- 외부 자료를 인용할 경우 가능한 범위에서 원문 출처를 함께 안내합니다.

## 3. 사용자 경험 기준

- 설명은 초보자도 이해할 수 있도록 단계적으로 작성합니다.
- 예제 코드/절차를 포함해 실제 적용 가능성을 높입니다.
- 광고보다 본문 정보 전달을 우선합니다.

## 4. 업데이트 및 정정 정책

- 내용 오류를 확인하면 게시글을 수정합니다.
- 기술 버전 변화로 정보가 오래된 경우 업데이트를 진행합니다.
- 중요한 정정이 있는 경우 본문에 변경 사실을 반영합니다.

## 5. 금지/제한 콘텐츠

- 저작권을 침해하는 무단 복제 콘텐츠
- 의미 없는 키워드 나열, 내용이 빈약한 자동 생성 페이지
- 사용자에게 오해를 줄 수 있는 과장/기만 표현

## 6. 문의 및 신고

오류 제보, 정정 요청, 저작권 관련 요청은 [문의하기](/contact/)를 통해 전달해 주세요.
