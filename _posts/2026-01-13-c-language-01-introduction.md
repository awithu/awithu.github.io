# C언어 강의 1편: C언어 소개 및 개발 환경 설정

## C언어란?

C언어는 1972년 벨 연구소의 데니스 리치(Dennis Ritchie)가 개발한 프로그래밍 언어입니다. 시스템 프로그래밍, 운영체제 개발, 임베디드 시스템 등에서 널리 사용되는 강력하고 효율적인 언어입니다.

## C언어의 특징

### 장점
- **빠른 실행 속도**: 컴파일 언어로 실행 속도가 빠름
- **메모리 제어**: 직접적인 메모리 관리 가능
- **이식성**: 다양한 플랫폼에서 사용 가능
- **기본 언어**: 많은 프로그래밍 언어의 기반

### 단점
- **메모리 관리**: 수동으로 메모리를 관리해야 함
- **복잡성**: 포인터 등 개념이 어려울 수 있음
- **디버깅**: 메모리 오류 디버깅이 어려움

## 개발 환경 설정

### Windows 환경

#### 1. MinGW-w64 설치
- MinGW-w64는 Windows에서 GCC 컴파일러를 사용할 수 있게 해주는 도구입니다
- 공식 사이트에서 다운로드: https://www.mingw-w64.org/

#### 2. Visual Studio Code 설치
- 에디터로 Visual Studio Code를 추천합니다
- C/C++ 확장 프로그램 설치

#### 3. 첫 번째 프로그램 작성

`hello.c` 파일을 만들고 다음 코드를 작성하세요:

```c
#include <stdio.h>

int main() {
    printf("Hello, World!\n");
    return 0;
}
```

#### 4. 컴파일 및 실행

```bash
gcc hello.c -o hello
./hello
```

출력 결과:
```
Hello, World!
```

## 프로그램 구조 이해하기

```c
#include <stdio.h>  // 표준 입출력 라이브러리 포함

int main() {        // 프로그램의 시작점
    // 코드 작성
    return 0;       // 프로그램 종료
}
```

- `#include <stdio.h>`: 표준 입출력 함수를 사용하기 위한 헤더 파일
- `int main()`: 프로그램의 진입점(Entry Point)
- `return 0`: 프로그램이 정상 종료되었음을 의미
